Command Prompt
==============

Installation
------------
1. Copy doskey.cmd into <code>C:/</code>

Setup Command Prompt desktop shortcut
-------------------------------------
1. Navigate to <code>C:\Windows\System32</code>
2. Create shortcut
	* Target:
		* <code>%windir%\system32\cmd.exe /K C:\doskey.cmd</code>
	* Start in:
		*<code>C:\git</code>
